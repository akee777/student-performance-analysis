# 📊 Student Performance Analysis

## 📌 Objective

Analyze student performance based on gender and scores.

## 🛠 Tools Used

* Python
* Pandas
* Matplotlib
* Seaborn

## 📈 Key Insights

* Female students have higher reading scores
* Male students have slightly higher math median
* Score distribution varies between genders
* Outliers exist in both groups

## 📂 Dataset

Student Performance Dataset
